多个git公钥秘钥管理

一个公钥秘钥就可以通行所有git仓库登录(或者服务器登录)并且还挺安全,但是有时候有创建多个key的需要，那怎么实现多个秘钥登录多台服务器呢

  • ssh-keygen -t rsa -C "账号绑定的邮箱" 回车之后会有路径起名字操作多个账号最好区分开(例如 id_rsa_work、id_rsa_private 等)
  • cd ~/.ssh cat id_rsa_private.pub 将输出公钥给到管理员添加到git后台
  • 在.ssh 文件夹下 touch config(创建 config 文件)vim config 按 a/insert 进入编辑模式，像下面这样输入
# gitlab                                // 注释
Host workGitLab                         // 起别名
    HostName gits.bakejinfu.com         // 仓库域名(不要有http://和末尾的/)
    PreferredAuthentications publickey  // 使用公钥
    IdentityFile ~/.ssh/id_rsa_work     // 该仓库对应的密钥地址
    User wangpeng                       // 登录用户名

# github
Host privateGitHub
    HostName github.com
    PreferredAuthentications publickey
    IdentityFile ~/.ssh/id_rsa_private
    User wp0051

# gitee
Host Gitee
    HostName gitee.com
    PreferredAuthentications publickey
    IdentityFile ~/.ssh/id_rsa_node
    User wp0051
    按 esc 然后输入 :wq 退出并且保存

  • 改变拉取项目命令或者修改远程仓库地址从git clone git@gitee.com:wp0051/front-note.git变成git clone git@Gitee:wp0051/front-note.git

  • 之后就是修改项目、git add .;git config user.name;git config user.email '';git commit -m '';git push
