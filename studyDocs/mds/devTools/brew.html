安装 homebrew

安装

搜索打开homebrew文档（https://docs.brew.sh/Installation），执行 /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

添加全局环境变量

编辑~/.bashrc(或者.zshrc)，添加

export PATH=/opt/homebrew/bin:$PATH

保存更改，执行 source .bashrc(.zshrc)

校验

brew --help

出现的问题及解决

Field to connect to raw.githubusercontent.com port 443

打开 https://www.ipaddress.com/ 输入访问不了的域名 查看给出的IP Address,打开

cd /etc/
sudo vim hosts  // 添加 `***.***.***.** raw.githubusercontent.com` ，保存

执行ping raw.githubusercontent.com,看是否能ping通，重写执行安装命令

Field to connect to github.com port 443

按照上一步办法执行

使用

node

brew install node

npm源切换工具

npm i -g nrm

nrm use taobao

nrm use npm

yarn

brew install yarn

MongoDB - 文档型数据库

brew tap mongodb/brew

brew install mongodb-community

brew services start mongodb-community

VSCode - 代码编辑器

brew cask install visual-studio-code

Chrome - 更适合开发的浏览器

brew cask install google-chrome

安装 iTerm2 - 更强大的终端工具，支持窗口分割，不过大多数直接用VSCode终端也够用

brew cask install iterm2
