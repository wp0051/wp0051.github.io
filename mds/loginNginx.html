登陆

密码登陆

  1. 在阿里云、腾讯云、华为云购买服务器。
  2. 在控制台，重置密码。
  3. ssh (-p 22)(如果端口是22，可省略) root@"服务器地址" 回车登陆,输入密码(也可以终端新建远程连接，把这个配置写进去)

ssh登陆

普通命令行登陆

  1. 本地生成密钥，公钥(已有跳过)
  2. "cat 公钥路径" 打开公钥，完整复制内容(或者把文件copy到服务器上面去: scp /path/filename root@servername:/path/)
  3. 密码登陆服务器，cd ~/.ssh(没有mkdir创建),vi authorized_keys(没有touch创建),按 “i”输入，将上面复制的公钥内容拷贝进去，esc 输入:wq保存并退出
  4. 这里，还需注意服务器上.ssh文件夹及authorized_key这个文件的访问权限。执行chmod 600 ~/.ssh/authorized_keys以及chmod 700 ~/.ssh
  5. 一切ok，本地终端直接运行下面的指令，不需要输入密码了，直接就登陆进去了！
ssh root@{HostName} -i ~/.ssh/id_rsa_personal   // 公钥密钥不是默认的id_rsa
ssh root@{HostName}   // 公钥密钥是默认的id_rsa

通过指定公钥配置登陆

  1. 上面第4部走完后，每次登陆都需要输入ip很麻烦，虽然可以通过mac终端新建远程，每次不需要输入，但是依旧需要记住ip地址，依旧不是非常方便，所以有了下面这种方法
  2. 打开本地的.ssh "cd ~/.ssh","vi config"(没有touch创建),输入
Host dev  // 随便取，尽量简单
 HostName 服务器地址
 User 登陆的用户名
 Port 端口号
 IdentityFile ~/.ssh/id_rsa // 密钥地址
  1. 一切ok，现在使用ssh dev，不需要输入密码了，不需要ip，直接就登陆进去了！
