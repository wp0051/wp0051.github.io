xshell

密码连接

1.xshell新建连接 起个名、选择ssh协议、输入要连接服务器的公网ip、端口号默认22 2.用户身份验证 选择password，输入用户名(默认root)、密码在阿里云服务器控制台里面重置一下密码设置密码，输入密码，连接即可

修改密码

1.使用需要修改密码的用户登录服务器，之后输入 passwd 设置新密码 再次输入(符合规定的密码)

密钥连接

1.在本地生成密钥和公钥(在./git.md里面有),当然也可以用已经有的，在服务器的.ssh文件夹(在服务器生成密钥公钥时会自动创建),用chmod 700 .ssh 设置ssh文件权限为700,chmod 600 .ssh/authorized_keys,设置他为600,vim authorized_keys,在本地cat id_rsa_.pub,将输出的东西，复制粘贴到authorized_keys中去，保存退出。 2.再vim /etc/.ssh/sshd_config 将 #RSAAuthentication yes、#PubkeyAuthentication yes、#AuthorizedKeysFile .ssh/authorized_keys 前面的#，含义分别是开启 RSA 密钥验证，公钥验证功能，设置公钥验证文件的路径，保存退出。 3.还可以把 PasswordAuthentication yes，yes 改成 no，禁止使用密码登录。 4.在打开这个连接的属性，在用户身份验证里面选择 Public Key 导入 选择本地 上面第2条里面添加到authorized_keys里面去的“公钥”对应的密钥(即上面是cat id_rsa_.pub这个“公钥”，那么现在导入的文件就是id_rsa_*这个对应的“密钥”),service sshd restart 重启服务，关闭连接，再次连接

*** service sshd restart 重启服务 ***

文件及文件夹删除

1.rm a.txt;删除文件 会有确认操作 2.rm -f a.txt;删除文件 不许确认删除文件 3.rm -r a;递归删除文件夹 每个文件确认一次 3.rm -rf a;删除文件夹 不需确认删除文件夹 4.ls -a;查看目录下文件(包括隐藏文件)

mysql安装及连接

mysql安装

1.创建文件夹、下载包 mkdir mysql;wget -i -c http://dev.mysql.com/get/(mysql57-community-release-el7-10.noarch.rpm) 括号里面的是可以替换的 2.浏览器打开 https://dev.mysql.com/downloads/repo/yum/ 选择下载版本 点击下载 进入页面 选择直接下载,不登陆 出现下载弹出框 把文件名 拷出来 把上面括号里的替换掉，就可以下载指定的mysql了 3.使用上面的命令就直接下载了安装用的Yum Repository，大概25KB的样子，然后就可以直接yum安装了。 4.yum -y install (mysql57-community-release-el7-10.noarch.rpm);安装 5.yum -y install mysql-community-server;安装mysql服务 6.设置登录密码 set password for root@localhost = password('123456'); 7.退出mysql, exit; 8.但此时还有一个问题，就是因为安装了Yum Repository，以后每次yum操作都会自动更新，需要把这个卸载掉：yum -y remove (mysql57-community-release-el7-10.noarch);(没有.rpm)

连接
