SQL Structure Query Language 结构化查询语言

面向集合的描述性非过程化语言 非过程性 大多数语句都是独立执行的与上下无关的。 从用途上分类 DQL：

  • 数据查询语言（select）
  • DML：数据操纵语言 update delete
  • DDL：数据的定义语言 create
  • DCL：数据控制语言 定义用户权限，修改提交方式 grant

MySQL

数据的操作

Create Read Update Delete

  • 插入数据 insert into 表名 (字段名称1,字段名称2) VALUES (值1,值2),(值1,值2);
  • 删除 delete from 表名 where 条件;
  • 更新数据 update 表名 set 字段名字=字段值,字段名字=字段值 where 条件;
  • 查询 SELECT * FROM 表名;
  • 查询某一列：SELECT name FROM students;
  • 查询多列：SELECT name,age FROM students;
  • 查询起别名：SELECT age AS a FROM students;
  • 查询结合运算 SELECT age+1 AS a FROM students;对表的数据没有任何影响
  • 使用聚合函数
  • MIN MAX COUNT AVG(average) SUM
  • 平均值SELECT AVG(age) AS avg_age FROM students;
  • 查询总人数SELECT COUNT(*) AS num FROM students;
  • 表名+字段 SELECT students.name FROM students;
  • 查询子句：
  • WHERE 判断过滤
  • 所有的：SELECT * FROM students WHERE 1;
  • 等值判断 SELECT * FROM students WHERE id=1;

  • < <>

  • AND SELECT * FROM students WHERE age=20 AND id<>4;
  • OR SELECT * FROM students WHERE age>20 OR id=2;
  • IN SELECT * FROM students WHERE id IN (2,3,4);
  • BETWEEN SELECT * FROM students WHERE age BETWEEN 20 AND 30;
  • 模糊查询 SELECT * FROM students WHERE name LIKE '王%';（%任意字符任意个数(王。。。。。。)，_任意一个字符（王。））
  • ORDER BY 排序方式
  • SELECT * FROM students ORDER BY age;(升序)ASC
  • SELECT * FROM students ORDER BY age DESC;(降序)
  • SELECT * FROM students WHERE age>20 ORDER BY age;
  • LIMIT 截取
  • SELECT * FROM students LIMIT 2,1;(从第二条开始后截取一条)
  • SELECT * FROM students LIMIT 1 OFFSET 2;(从第2条后取1条)
  • SELECT * FROM students WHERE age>20 ORDER BY age LIMIT 0,2;
  • GROUP BY 分组查询 结合聚合函数
  • SELECT COUNT(*) AS num,age FROM students GROUP BY age;
  • SELECT age,COUNT(*) AS num FROM students WHERE name LIKE "王%" GROUP BY age ORDER BY num LIMIT 0,2;
  • HAVING 对于分组之后的结果进行过滤
  • SELECT age,COUNT(*) AS num FROM students GROUP BY age HVAING num>1
  • WHERE -> GROUP BY -> HAVING ->ORDER BY -> LIMIT
  • 多表查询
  • SELECT * FROM students,class; 笛卡尔积
  • 连接查询
  • 左连接 以左表为基础 SELECT * FROM students LEFT JOIN class ON students.cid=class.cid;
  • 右连接 以右表为基础 SELECT * FROM students RIGHT JOIN class ON students.cid=class.cid;
  • 内连接 只会查到有关联的数据 SELECT * FROM students INNER JOIN class ON students.cid=class.cid;
  • 内连接 SELECT * FROM students,class WHERE students.cid=class.cid;
  • 自连接 SELECT c1.cid,c1.catname,c1.sort,c1.pid FROM category c1 LEFT JOIN category c2 ON c1.cid=c2.pid;
  • 查询学生的姓名，年龄，id和他所在的班级名称
  • SELECT students.id,students.name,students.age,class.name AS classname FROM students,class WHERE students.cid=class.cid;
  • SELECT s.id,s.name,s.age,c.name AS classname FROM students s,class c WHERE s.cid=c.cid;
  • 查询每个班级有多少人
  • SELECT c.name,count(*) FROM students s,class c where s.cid=c.cid GROUP BY c.name;
